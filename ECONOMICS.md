# ECONOMICS.md

> Unit economics, revenue model, and financial projections for SpendScope.

---

## Business Model

SpendScope is currently **free** (audit engine, shareholder URLs).

Potential monetization (future):
1. **Pro Reports** — Deeper PDF reports with benchmarks ($9 one-time)
2. **Team Plans** — Ongoing monitoring + alerts ($29/mo)
3. **API Access** — Audit API for tooling companies

---

## Unit Economics (Projections)

### Free Tier Cost Estimate (per audit)

| Component | Est. Cost |
|---|---|
| Supabase read/write | ~$0.000 (free tier) |
| Claude API (summary) | ~$0.002 (500 tokens) |
| Vercel hosting | ~$0.000 (free tier) |
| **Total per audit** | **~$0.002** |

### Break-Even (Pro Reports model)

| Metric | Value |
|---|---|
| Price per report | $9 |
| Claude cost per report | $0.002 |
| Margin | ~99.9% |
| Break-even audits/month | 1 paid conversion |
